# Investor onboarding SOP (private beta)

## 1. Pre-qualify
- Confirm accredited / firm fit before invite

## 2. Invite
- Generate `/auth/sign-up?role=investor` from system health tools
- Enable `PRIVATE_BETA_MODE` so signup copy reflects approval requirement

## 3. Profile submission
- Investor completes `/investor/onboarding`
- Submits for staff review (`approval_status = submitted`)

## 4. Staff approval
- [ ] Review `/admin/investors`
- [ ] Approve, reject, or request changes
- [ ] Do not approve without profile completeness check

## 5. Post-approval
- [ ] Investor can access `/investor/opportunities`
- [ ] Document access only with company relationship
- [ ] Create deal room for structured diligence when ready

## 6. Activation milestones (beta operations)
Signed up → Staff approved → First opportunity view → First watchlist save → First interest → First deal room interaction

## 7. Support
- **Copy login link** from beta operations founder/investor rows
- Feedback via investor settings
