# Founder onboarding SOP (private beta)

## 1. Invite
- Generate link from `/admin/system-health` → Beta onboarding tools
- Send to curated founder only; log externally

## 2. Signup & profile
- Founder completes `/auth/sign-up?role=founder`
- Redirect to `/founder/onboarding`

## 3. Required steps
- [ ] Company profile complete
- [ ] Funding information
- [ ] Pitch deck / documents uploaded (`/founder/documents`)
- [ ] Readiness checklist reviewed (`/founder/readiness`)

## 4. Staff review
- [ ] Open `/admin/companies/[companyId]`
- [ ] Approve or request changes before marketplace publication
- [ ] Create deal room when investor diligence begins

## 5. Activation milestones (tracked in beta operations)
Signed up → Onboarding complete → First document → First lesson → First investor interaction → First deal room response

## 6. Support
- Use **Remind onboarding** if stalled >7 days
- Collect feedback via founder settings → Submit beta feedback
